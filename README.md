SocialGoal

Reference web app for ASP.NET MVC 5, EF 6 Code First, TDD and DDD. 

Technologies
============
* ASP.NET MVC 5
* EF 6 Code First 
* AutoMapper
* Autofac
* NUnit
* Moq

Patterns & Practices
===================
* Domain Driven Design (DDD)
* Test-Driven Development (TDD)
* Repository Pattern & Generic Repository
* Unit of Work 
* Dependency Injection
